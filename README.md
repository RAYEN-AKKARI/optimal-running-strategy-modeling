# Optimal Running Strategy Modeling

## Overview

This project explores how mathematical modeling and optimization can be used to determine optimal pacing strategies for running events.

The study is based on the classical Keller model and several extensions that incorporate physiological constraints, fatigue, energy availability, and stochastic variability.

The main objective is to understand how an athlete should distribute effort and speed throughout a race to minimize completion time.

## Main Topics

- Optimal pacing strategies
- Mathematical optimization
- Differential equations
- Energy constraints
- Fatigue modeling
- Stochastic performance models
- Comparison with running records

## Models Studied

- Keller model
- Mathis fatigue model
- Woodside long-distance extension
- Behncke bioenergetic model
- Whitt stochastic model
- Aftalion–Bonnans energy model
